# Kuma Book
Kuma Books is a forum that unites book readers and exchange their opinions about books that have been read

## Installation

Clone this repo

```bash
https://github.com/kubil-ismail/kuma_frontend.git
```
Intall dependency.

```bash
cd kuma_frontend
npm install
npm start
```
